
소스코드는 아래 사이트에서 가져왔습니다.
from https://github.com/sarbian/KerbalFontProject



한글폰트 생성방법은 아래와 같습니다.

1. unity 5.2.x 버전 설치

2. Text Mesh Pro_1.0.54.52 설치

3. asset bundle creator 설치

https://github.com/NiklasBorglund/AssetBundleCreator

4. 아래 폴더 추가 생성

\Unity Projects\New Unity Project 5\Assets\AssetBundles
\Unity Projects\New Unity Project 5\Assets\Data\AssetBundles\NotoSansCJKkr-Regular
\Unity Projects\New Unity Project 5\Assets\Editor
\Unity Projects\New Unity Project 5\Assets\UndoManager

5. 폰트 에셋 생성 및 저장

메뉴 > Window > TextMeshPro - Font Asset Creator 실행

\Unity Projects\New Unity Project 5\Assets\Data\AssetBundles\NotoSansCJKkr-Regular 에 저장

6. AssetBundleCreator 실행

메뉴 > Assets > BundleCreator > Asset Bundle Creator

실행하면 NotoSansCJKkr-Regular.unity3d 가 생성된다.
